# AIONIX Frontend

React + TypeScript landing page scaffolded with Vite.

## Structure

```txt
src/
  components/   Shared UI such as header and brand mark
  data/         Static page content and navigation data
  pages/        Route-level page composition
  sections/     Large page sections such as the hero
  styles/       Global styling and design tokens
```

## Commands

```bash
npm install
npm run dev
npm run build
```
